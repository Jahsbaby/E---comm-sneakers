# 🛍️ E-commerce Product Page

This is a solution to the **E-commerce Product Page** challenge on [Frontend Mentor](https://www.frontendmentor.io).  
The goal of this project was to build a fully responsive and interactive e-commerce product page with a dynamic cart and image gallery.

---

## 🚀 Features

- **Product Image Gallery** – Thumbnail selection with a lightbox modal for larger previews.  
- **Cart Functionality** – Add, remove, and update product quantities dynamically.  
- **Responsive Design** – Optimized layouts for mobile and desktop views.  
- **Interactive UI** – Smooth transitions, hover effects, and a modern feel.  
- **React Component Structure** – Clean and reusable component-based architecture.  

---

## 🛠️ Built With

- **React (JSX)** – For building interactive UI with reusable components.  
- **Custom CSS** – For pixel-perfect styling, responsiveness, and design fidelity.  
- **Frontend Mentor Assets** – Starter files and design guidelines.  

---

---

## 🔧 Installation & Setup

Clone this repository and run locally:

```bash
# Clone repo

## 🎨 Design & Functionality

* **Navbar**
  Contains logo, navigation links, and a cart icon.

* **Product Section**

  * Image gallery with thumbnails.
  * Enlarged product preview with lightbox modal.
  * Product details (title, description, pricing).
  * "Add to Cart" button with quantity selector.

* **Cart Dropdown**

  * Displays selected items.
  * Update or delete items dynamically.

* **Responsiveness**
  Layout adapts smoothly to desktop, tablet, and mobile devices.
